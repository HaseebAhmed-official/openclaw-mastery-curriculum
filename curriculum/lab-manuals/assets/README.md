# Screenshot Asset Tree

## Purpose

This directory stores reference screenshots for the classroom lab manuals.

## Subdirectories

- `core/` for phase A and B labs
- `advanced/` for phase C labs
- `specialization/` for phase D labs

## Source of truth

Use the [Reference Screenshot Manifest](../reference-screenshot-manifest.md) for exact filenames and required capture slots.

## Redaction rule

Do not commit screenshots that expose secrets, tokens, personal identifiers, or institution-specific private data.
